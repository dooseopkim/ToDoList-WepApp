# ToDoList-WepApp

HTML, CSS, Javascript만으로 만든 ToDoList-WebApp입니다.  
&nbsp;  
사용자의 이름과, 위치정보, ToDoList는 LocalStorage에 저장됩니다.  
위치정보를 https://openweathermap.org/api 로 요청하여 기온정보를 응답받아 화면에 출력합니다.
